# Retro Game Station

**Author:** Cojocaru Iulia-Alexandra

## Overview
**Retro Game Station** is a Python-based graphical application that allows users to enjoy three classic retro games:  
- **Highway Run**  
- **Pong**  
- **Snake**  

The project focuses on enhancing the graphical and auditory experience of these games through multimedia elements, without altering their core mechanics or logic.

---

## Features
- **Interactive User Interface**: A user-friendly menu for navigating and selecting games.
- **Multimedia Enhancements**:
  - JPEG and PNG images
  - WAV and MP3 audio files
  - Custom fonts
  - Smooth animations and sprite effects
- **Event-driven Logic**:
  - Sound effects and music triggered by specific actions.
  - Scrolling effects and dynamic intro animations.
- **Randomized Elements**:
  - Random positions in games like Snake and Highway Run.

---

## Technical Details
### Libraries Used
- **Pygame**: For graphical rendering, multimedia handling, animations, and event detection.
- **sys**: For system operations like closing windows.
- **os**: To manage file paths.
- **time**: To add delays in animations and displays.
- **random**: For generating random positions in games.

### Implementation Highlights
1. Developed a unified platform for all games with a cohesive design.
2. Implemented:
   - Main menu and game selection screens.
   - Intro screen animations and scrolling effects.
   - Back buttons and game selection system.
3. Balanced memory usage with optimized multimedia integration.
